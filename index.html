Tu Rutina de Ejercicios Futurista

Día 1: Upper Body Push

Calentamiento (5-10 minutos):

  • Trote ligero en el lugar
  • Círculos con los brazos
  • Rotaciones de hombros
  • Flexiones (5-10 repeticiones)

Rutina Principal:

1. Press de banca con barra - 3 series de 8-10 repeticiones

   Posición inicial          Posición final
       _____                     _____
      |     |                   |     |
      | ___ |                   | ___ |
   O--|-----|--O             O--|-----|--O
  /|\                           |\___/|
  / \                           |  |  |
 ___________               ___________
|___________|             |___________|

Acuéstate en el banco, agarra la barra un poco más ancho que el ancho de los hombros. Baja la barra hasta el pecho y luego empuja hacia arriba.

2. Press militar de pie - 3 series de 8-10 repeticiones

 Posición inicial      Posición final
     O------O              |---|
     |      |              |   |
    /|\    /|\            _O---O_
     |      |            / \   / \
    / \    / \          |   | |   |
                        |   | |   |

De pie, agarra la barra a nivel de los hombros y presiónala por encima de la cabeza. Baja controladamente de vuelta a los hombros.

3. Aperturas con mancuernas en banco inclinado - 3 series de 10-12 repeticiones

 Posición inicial      Posición final
     \   /                  | |
      \ /                   | |
   O---O---O            O---O---O
    \     /              \     /
     \___/                \___/
    /  |  \               /   \
   /   |   \             /     \

En banco inclinado, baja las mancuernas hacia los lados, manteniendo un ligero doblez en los codos, y luego vuelve a subirlas.

4. Fondos de tríceps - 3 series de 10-15 repeticiones

 Posición inicial      Posición final
   |---|   |---|        |---|   |---|
     O       O            O       O
    /|\     /|\          /|\     /|\
   | | |   | | |        _| |_   _| |_
    / \     / \        |  |  | |  |  |
                       |  |  | |  |  |

Usa barras paralelas o el borde de un banco estable. Baja tu cuerpo flexionando los codos y luego empuja hacia arriba.

5. Flexiones - 3 series hasta el fallo

 Posición inicial      Posición final
    O       O             O---O
   /|\     /|\           /     \
  | | |   | | |         |   |   |
 /  |  \ /  |  \       /    |    \
                      |     |     |

Comienza en posición de plancha, baja el pecho al suelo manteniendo el cuerpo recto, y luego empuja de vuelta a la posición inicial.

Enfriamiento (5-10 minutos):

Estiramientos ligeros para pecho, hombros y tríceps.

Completado:

Día 2: Lower Body

Calentamiento (5-10 minutos):

  • Caminata rápida o jogging suave
  • Saltos de tijera
  • Sentadillas sin peso
  • Estiramiento dinámico de piernas

Rutina Principal:

1. Sentadillas con barra - 4 series de 8-10 repeticiones

 Posición inicial      Posición final
    |-----|              |-----|
      \ /                  \ /
     O-O-O                O-O-O
      |                    |
     /|\                  / \
    | | |                |   |
   /  |  \              /     \
  |   |   |            |       |

Coloca la barra en los trapecios, baja flexionando rodillas y caderas, luego empuja hacia arriba.

2. Peso muerto - 3 series de 8-10 repeticiones

 Posición inicial      Posición final
       O                 |-----|
      /|\                  O
     | | |                /|\
    /  |  \              | | |
   |   |   |            /  |  \
  |-------|           |    |    |

Con la barra en el suelo, flexiona caderas y rodillas para agarrarla, luego levanta extendiendo piernas y espalda.

3. Lunges con mancuernas - 3 series de 12 repeticiones por pierna

 Posición inicial      Posición final
     O   O               O   O
     /|\ /|\             /|\_/|\
   | | | | |           | |   | |
  /  | |  | \         /  |___|  \
 |   | |  |  |       |   |   |   |
     | |  |              |   |

Da un paso adelante, baja la rodilla trasera cerca del suelo, luego empuja de vuelta a la posición inicial.

4. Prensa de piernas - 3 series de 12-15 repeticiones

 Posición inicial      Posición final
    |_______|            |_______|
   /    O    \          /         \
  |    /|\    |        |     O     |
  |   | | |   |        |    /|\    |
  |  /  |  \  |        |   | | |   |
  | |   |   | |        |  /  |  \  |
   \|___|___| /         \|___|___|/

En la máquina, empuja la plataforma con los pies, extendiendo las piernas, luego regresa controladamente.

5. Elevaciones de pantorrilla - 3 series de 15-20 repeticiones

 Posición inicial      Posición final
      | |                  O O
      | |                 /|/|\
      O O                / | | \
     /|\|\               | | | |
    | | | |              | | | |
   /  | |  \             | | | |
  |   | |   |            | | | |
      ___                  ___

De pie con las puntas de los pies en un escalón, levanta los talones lo más alto posible, luego baja.

Enfriamiento (5-10 minutos):

Estiramientos estáticos para cuádriceps, isquiotibiales, glúteos y pantorrillas.

Completado:

Día 4: Upper Body Pull

Calentamiento (5-10 minutos):

  • Rotaciones de brazos
  • Elevaciones de hombros
  • Flexiones de pared
  • Tracciones asistidas o negativas

Rutina Principal:

1. Tracciones o pull-ups - 3 series hasta el fallo (o asistidas si es necesario)

  |---|---|---|       |---|---|---|
     \  O  /             _\O/_
      \/|\\/     ->       |
       / \                |
                         / \

Cuelga de la barra y tira de tu cuerpo hacia arriba hasta que tu barbilla esté por encima de la barra.

2. Remo con barra - 3 series de 8-10 repeticiones

      O                O
     /|\    ->        /|\_____
    / | \            / |
   /  |  \          /  |
 ==========================================

Inclínate hacia adelante con la barra colgando, tira la barra hacia tu abdomen inferior.

3. Face pulls - 3 series de 12-15 repeticiones

    O        O---O      \O/
   /|\  ->  /     \  ->  |
   / \     |   |   |    / \
          /    |    \

Con una polea a la altura de la cara, tira de la cuerda hacia tu frente, separando los brazos al final.

4. Curl de bíceps con barra - 3 series de 10-12 repeticiones

     O             O
    /|\    ->     /|\
    _|_           |O|
   |   |         | | |
   |___|         |___|
    / \           / \

De pie, levanta la barra hacia los hombros flexionando los codos, luego baja controladamente.

5. Remo unilateral con mancuerna - 3 series de 10 repeticiones por brazo

    O               O
   /|    ->        /|\_
  | |             | | 
 /__|\\           /__|\\
    |               |

Con una mano y rodilla sobre un banco, tira de la mancuerna hacia tu cadera.

Enfriamiento (5-10 minutos):

Estiramientos para espalda, bíceps y hombros.

Completado:

Día 5: Full Body & HIIT

Calentamiento (5-10 minutos):

  • Saltos de cuerda
  • Jumping jacks
  • Burpees suaves
  • Rotaciones de tronco

Rutina Principal (Circuito HIIT - 4 rondas, 40 segundos de trabajo, 20 segundos de descanso):

1. Burpees con flexiones

 O    _O_    /\     O     O---O
/|\  _/ \_   ||    /|\     \   \
/ \         / \   / \  ->   \___\

Realiza un burpee estándar, añadiendo una flexión cuando estés en el suelo.

2. Sentadillas con salto

 O        O      _O_
/|\  ->  / \  ->  |
/ \     |   |    / \
       /     \

Haz una sentadilla profunda y salta explosivamente al subir.

3. Mountain climbers

 O           O          O
/|\_____    /|\_____   /|\_____
 \           \        /

En posición de plancha, alterna llevando las rodillas hacia el pecho rápidamente.

4. Peso muerto con mancuernas y remo

   O              O             O
  /|\    ->      /|\    ->     /|\__
 | | |          | | |         | | |
/  |  \        /__|__\       /  |  \
   |              |             |

Realiza un peso muerto y añade un remo al final del movimiento.

5. Plancha con toques de hombro

   O              O              O
  /|\____    ->  /|\_O__    ->  /|\____
 / \            / \            / \

En posición de plancha, toca tu hombro opuesto con cada mano alternativamente.

6. Saltos laterales

    O      _O_      O
   /|\  ->  |  ->  /|\
   / \     / \    / \
  |   |            |   |

Salta de lado a lado, aterrizando suavemente y cambiando de dirección rápidamente.

Rutina de Core (3 rondas):

1. Crunches - 20 repeticiones

2. Russian twists - 30 segundos

3. Plancha - 45 segundos

Enfriamiento (5-10 minutos):

Estiramientos suaves para todo el cuerpo, enfocándote en las áreas más trabajadas.
